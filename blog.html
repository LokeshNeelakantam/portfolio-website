How I Built My Resume Website Using HTML, CSS and JavaScript

In this blog, I explain how I built my personal resume website using HTML, CSS, and JavaScript with dark mode and responsive design.

Read More

Introduction

I wanted to show my skills in a better way than only a PDF resume. So I created my own resume website using HTML, CSS and JavaScript. In this blog, I explain how I built this website and what I learned.

Technologies Used

HTML, CSS, JavaScript

Website Sections

My resume website contains sections like About, Skills, Experience, Projects, Education and Certifications. This makes the website easy to understand for recruiters.

Navigation Bar and Smooth Scroll

I created a navigation bar at the top. When I click on a menu item, the page scrolls smoothly to that section using JavaScript.

Glassmorphism Design

I used a glass-style design with blur effect and transparent background to make the website look modern and professional.

Dark Mode Feature

I added a dark mode button. When the button is clicked, the website switches between dark and light mode using JavaScript.

Responsive Design

I used media queries so the website works well on mobile, tablet and desktop screens.

What I Learned

This project helped me understand HTML structure, CSS styling, JavaScript interaction, and responsive web design.

Conclusion

Building this resume website improved my frontend skills and helped me present my profile professionally.
